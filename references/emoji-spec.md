# emoji 图标规范

## 标准

| 标题级别 | emoji | 必须 |
|---------|-------|------|
| `h1`（`#`） | 🛠️ 等项目相关 emoji | 必须 |
| `h2`（`##`） | 🎯 / ✨ / 🚀 / 📦 / 📁 / ✅ / 🔗 / 📜 | 必须 |
| `h3`（`###`） | 🔧 / 📌 / ⚠️ | 按需 |

## 规则

- 每个 `##` 章节都必须有 emoji，不能裸标题
- emoji 放在标题最前面，如 `## 🎯 触发条件`
- 常见组合：`## 🎯 触发条件`、`## ✨ 功能特性`、`## 🚀 快速开始`、`## 📦 安装`、`## ✅ 安装后验证`、`## 🤝 欢迎贡献`、`## 📜 许可证`
