# shields.io 徽章 URL 格式规范

## 徽章类型

| 类型 | shields.io URL 格式 |
|------|---------------------|
| License | `![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)` |
| Version | `![version](https://img.shields.io/badge/version-X.Y.Z-green.svg)` |
| Platforms | `![platforms](https://img.shields.io/badge/platforms-Hermes%20Agent%20%7C%20Claude%20Code-blue.svg)` |
| Category | `![category](https://img.shields.io/badge/category-{category}-blue.svg)` |
| Stars | `![stars](https://img.shields.io/github/stars/{user}/{repo}?style=flat&label=Stars)` |

## 常见问题

- URL 中空格用 `%20` 或 ` `（空格）均可
- 颜色可选：green / blue / yellow / red / orange
- `[label]` 中不能含空格（用 `-` 替代）
